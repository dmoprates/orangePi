# Orange Pi

## Sobre o repositório

Este repositório será utilizado para armazenar os comandos em bash para implementar na Orange Pi
visando realizar toda a configuração necessária para colocá-la em produção.

Os códigos serão utilizados para manter o ambiente em caso de atualização do Linux por parte do
desevolvedor ou em caso de necessidade de formatação do cartão SD.

A intenção do projeto é criar um home lab utilizando a Orange Pi.

No momento da construção desse repositório está sendo utilizado o seguinte SO:

* Orangepi4pro_1.0.6_debian_bookworm_desktop_xfce_linux5.15.147
